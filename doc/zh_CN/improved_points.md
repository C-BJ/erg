
## 可变对象和不变对象区别明显

Python 的可变对象和不变对象、堆对象和值对象之间的区别并不明显，因此，我们需要记住一些知识，比如元组是不变的，但列表是可变的……另外，当你想让自己的班级保持不变时，你必须遵循繁琐的步骤。


```python
# このコードが過去のPythonでは有効だったと信じられますか?
i = 256
assert i is 256
i = 257
assert i is not 257
```
