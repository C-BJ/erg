# Erg存储表结构

```console
  └─┬ assets：图片等
    ├─ CODE_OF_CONDUCT：行为准则
    ├─┬ compiler
    │ ├─ erg_common：通用工具
    │ ├─ erg_compiler
    │ └─ erg_parser：解析器
    ├─┬ doc: 文档
    │ ├─┬ CN
    │ │ ├─ API：Erg标准API
    │ │ ├─ compiler：关于编译器实现
    │ │ ├─ dev_guide：开发者和贡献者指南
    │ │ ├─ python：Erg开发必备的Python知识
    │ │ ├─ syntax：Erg语法
    │ │ └─ tools：Erg命令行工具
    │ └─┬ JA
    │ ...
    ├─ examples：示例代码
    ├─ library：Erg脚本库
    ├─ src：main.rs和驱动所在目录
    └─ tests：测试代码
```
