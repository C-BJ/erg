# Tuple T: ...Type

包含多種類型對象的集合

## methods

* zip self, other

    組合兩個有序集合（數組或元組）

    ```erg
    assert ([1, 2, 3].zip [4, 5, 6])[0] == (1, 4)
    ```

* zip_by self, op, other

    一種泛化 zip 的方法。您可以指定一個二進制操作來組合
     `()`、`[]`、`{}`、`{:}` 也可以指定為運算符，分別生成元組、數組、集合和字典
    
    ```erg
    assert ([1, 2, 3].zip([4, 5, 6]))[0] == (1, 4)
    assert ([1, 2, 3].zip_by((),  [4, 5, 6]))[0] == (1, 4)
    assert ([1, 2, 3].zip_by([],  [4, 5, 6]))[0] == [1, 4]
    assert ([1, 2, 3].zip_by({},  [4, 5, 6]))[0] == {1, 4}
    assert ([1, 2, 3].zip_by({:},  [4, 5, 6]))[0] == {1: 4}
    assert ([1, 2, 3].zip_by(`_+_`, [4, 5, 6]))[0] == 5
    ```
