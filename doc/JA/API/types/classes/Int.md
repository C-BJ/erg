整数(Integer)を表すクラスです。インスタンスは0, 1, -1, 300000などがあります。
多くの言語では自然数を表す場合でもInt(に相当する型)を使用しますが、Ergではuse smaller typesの原則により、
Nat, NZInt, Interval型などを使用することが推奨されます。
IntはPythonに由来する型であり、その上位型には`Object`のみを持ちます。
