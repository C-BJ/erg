## debug

デバッグモードにする。これにより、Erg内部での挙動が逐次ログ表示される。
Rustの`debug_assertions`フラグとは独立。

## japanese

システム言語を日本語にする。
Erg内部のオプション、ヘルプ(help, copyright, licenseなど)、エラー表示は日本語化が保証される。

## simplified_chinese

システム言語を簡体字中国語にする。

## traditional_chinese

システム言語を繁体字中国語にする。
