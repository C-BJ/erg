## 常規

尊重他人

即使技能上有差異，也不要說刻薄的話或表示輕蔑

不要在分歧中咄咄逼人

不要否認對方的任何身份

不要發布任何嚴重冒犯他人的內容

不要發布垃圾郵件，釣魚消息或惡意挑釁的帖子

如果您發現有人違反了本行為準則，請立即聯繫 [管理員](mailto:moderation.erglang@gmail.com)。我們將採取適當的行動
